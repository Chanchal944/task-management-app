Page 2: Project Objective
The objective of this mini project is to develop a clean, responsive, and functional task management web application that allows users to:
Add daily tasks
Mark tasks as completed
Delete tasks
Filter tasks (All / Completed / Pending)
View task statistics
This project evaluates:
React fundamentals
Component-based design
State management
UI responsiveness
Deployment capability
Page 3: Problem Statement
Professionals and students often struggle with managing daily tasks efficiently. Many available tools are complex or overloaded with unnecessary features.
Proposed Solution
A lightweight, minimal, fast task management web app that:
Works instantly
Has simple UI
Requires no login
Provides clear task tracking
